# 🕉️ Umarudra Koteswara Swamy Temple Website

A modern and responsive temple website dedicated to **Umarudra Koteswara Swamy**, designed to provide devotees with complete information about the temple, including history, poojas, timings, festivals, and more.

---

## 🌟 Features

- 🏛️ Temple History & Spiritual Significance  
- 🕰️ Daily Timings & Darshan Details  
- 🎉 Festival Information  
- 🙏 Special Poojas & Services  
- 💰 Donation Page  
- 🗓️ Festival Calendar  
- 📢 Daily Updates Section  
- 🌐 Multilingual Support (English & Telugu)  
- 📱 Fully Responsive Design (Mobile + Desktop)  
- 🍔 Hamburger Menu for Mobile Navigation
